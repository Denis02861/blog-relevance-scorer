# Контекст вашего блога (для LLM-оценки трендов)

Заполните под себя. Этот файл уходит в LLM как system prompt.

## Кто я
[1-2 строки про себя: имя, чем занимаюсь, где блог, сколько подписчиков]

## Темы которые залетают (3-6 углов)
Тренд релевантен если попадает в любой из этих углов.
1. [например: переходы из найма в своё]
2. [...]

## Что НЕ работает (низкие лайки)
Эксперт-стиль («10 фишек»), длинные инструкции, пафос, мотивашка, лайфстайл.

## Голос
[3 строки: короткие предложения / эмоция в открывалке / финал-вопрос / самоирония]

## Шкала оценки
- 9-10: 2+ углов одновременно
- 7-8: один угол сильно
- 5-6: один угол слабо, требует натяжки
- 0-4: вне углов

## Angle — что это
Одна короткая строка с направлением зеркаления. БЕЗ выдуманных имён, цифр, сцен.

## Стоп-слова
[Чего не упоминать публично — названия компаний, города, имена]
