# Как сделать у себя такого «оракула»

Этот инструмент каждый день ищет вирусные посты в Reddit и Threads и говорит — какие из них могут «залететь» именно у вас, если их зеркалить. Ниже — простыми словами, что делать.

## Кому подойдёт

- Если вы хоть раз запускали что-то в терминале — справитесь за час.
- Если терминал — это страшное слово, найдите знакомого айтишника. Покажите ему этот README — он соберёт за 30 минут.

## Что вам понадобится

1. **Компьютер** с интернетом. Python 3.10+ должен быть установлен (Mac/Linux — обычно уже стоит, Windows — скачайте с python.org).
2. **Два «ключа доступа»** — это как пароли, которые выдают сайты, чтобы программа могла обращаться к их ИИ:
   - **OpenAI**: зайдите на platform.openai.com → создайте аккаунт → раздел «API keys» → «Create new secret key» → скопируйте. Положите $5 на баланс — этого хватит на полгода работы.
   - **OpenRouter**: openrouter.ai → войти через Google → раздел «Keys» → «Create Key» → скопируйте. Положите $5.
3. **Свои опубликованные посты** — те которые уже выкладывали в свой блог. Чем больше — тем точнее работает.

## Пошагово

### Шаг 1. Скачайте код
Скачайте этот репозиторий: нажмите зелёную кнопку «Code» → «Download ZIP» → распакуйте.

### Шаг 2. Поставьте библиотеки
Откройте терминал в папке с распакованными файлами и введите одну строку:
```
pip install openai requests numpy
```
Подождите минуту.

### Шаг 3. Положите свои посты
Создайте в папке подпапку `my_posts`. Туда положите каждый ваш пост отдельным текстовым файлом (.md или .txt). Например:
- my_posts/post-1.md
- my_posts/post-2.md
- ...

### Шаг 4. Заполните контекст о себе
Откройте файл `context-template.md`, переименуйте в `context.md`, и впишите про себя:
- Кто вы, чем занимаетесь
- Какие 3-6 тем у вас залетают
- Какой у вас голос (короткие предложения / с эмоциями / с юмором)
- Стоп-слова — что нельзя упоминать публично

Это самое важное — ИИ будет ориентироваться на этот файл при оценке.

### Шаг 5. Подключите ключи и запустите
В терминале:
```
export OPENAI_API_KEY=ваш-ключ-от-openai
export OPENROUTER_API_KEY=ваш-ключ-от-openrouter
python relevance_layer.py
```

### Откуда брать кандидатов (виральные посты других)?
- Самое простое: руками копируйте 10-20 свежих виральных постов из Reddit/Threads в файл `candidates.json` в формате `[{"text": "...", "likes": 123, "comments": 45}, ...]`
- Сложнее: подключить автоматический сборщик через Reddit API или парсер Threads. Это уже задача для разработчика — но можно поставить программисту техзадание в 2 строки.

## Что получите на выходе
Список постов с оценкой 0-10, где 10 — «точно зайдёт». На каждый — короткий «угол»: как переписать под себя.

Пример:
```
8/10 — переход из найма в своё, через свою личную историю
   reasoning: попадает в 2 угла, эмоция есть, формат рабочий
```

## Сколько стоит
Около 30 рублей в месяц если запускать каждый день. Главный расход — ИИ-оценка (Haiku 4.5 — самая дешёвая из топовых).
